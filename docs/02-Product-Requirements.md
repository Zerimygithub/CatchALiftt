# CatchALiftt Product Requirements

## Overview

CatchALiftt is an AI-powered ride-sharing platform that connects Ride Hosts and Travellers for daily commuting.

The objective is to make commuting:

- Affordable
- Convenient
- Safe
- Intelligent
- Eco-friendly
