# CatchALiftt

**Every Journey Feels Lighter**

An AI-powered ride-sharing platform designed to connect Ride Hosts and Travellers with intelligent route matching, minimal detours, and a premium user experience.

---

## 🚧 Project Status

Currently under active development.

Current Version: **Prototype 7**

---

## ✨ Vision

CatchALiftt aims to make daily commuting smarter, safer, and more efficient by helping travellers join existing rides instead of creating new ones.

The platform focuses on:

- AI-assisted ride matching
- Smart pickup suggestions
- Minimal host detours
- Wallet & secure payments
- Live tracking
- Rewards & referrals
- Safety and trust
